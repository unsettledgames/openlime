Class: BoundingBox

BoundingBox

The bounding box is a rectangular box that is wrapped as tightly as possible around a geometric element. It is oriented parallel to the axes. It is defined by two opposite vertices. The class It includes a comprehensive set of functions for various processing tasks related to bounding boxes.
