Class: Skin

Skin

The static class Skin implements some utilities for handling the skin file. A skin file is a SVG file containing SVG icons that are used by *UIBasic for customizing the visual appearance of the user interface (i.e., icons for buttons, menu, toolbar, dialog...). Each SVG drawing element must be tagged with a 'class' attribute whose name must begin with openlime-: for instance, the HOME icon is a SVG element tagged with class="openlime-home".
